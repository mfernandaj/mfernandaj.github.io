Vamos a crear nuestro primer componente en Angular que mostrará el clásico "Hola Mundo". Este es un ejemplo sencillo pero fundamental para entender cómo funciona Angular.

  1. Asegúrate de tener instalado Angular CLI. Puedes verificarlo usando: 
    ng --version
  2. Crea un nuevo proyecto Angular con el siguiente comando: 
    ng new hola-mundo
  3. Una vez creado el proyecto, navega a la carpeta y ejecuta el servidor local: 
    cd hola-mundo
    ng serve
    Abre tu navegador en http://localhost:4200
    .
  4. Edita el archivo src/app/app.component.ts, el template y reemplaza o comenta su contenido con: 
    <h1>Hola Mundo con Angular!</h1>
  5. Guarda los cambios y actualiza tu navegador. Deberías ver el mensaje "¡Hola Mundo con Angular!" en pantalla.
