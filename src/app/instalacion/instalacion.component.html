Aprende cómo instalar y configurar Angular paso a paso. En esta sección te guiaré en el proceso de instalación de Angular para Windows. Siguiendo los siguientes pasos:

Paso 1. Requisitos previos: Instalar Node.js y npm

Asegúrate de contar con lo siguiente antes de instalar Angular:

Node.js proporciona un entorno para ejecutar JavaScript fuera del navegador, mientras que npm (Node Package Manager) permite gestionar las dependencias del proyecto, como Angular CLI.

Es fundamental asegurarte de que tienes instaladas versiones compatibles:

  • Node.js: versión 14.20.0 o superior.
  • npm: versión 6.14.0 o superior.
  • Node.js y npm. Descárgalos desde la página oficial de Node.js.
  • Verifica su instalación con los comandos:
    • node -v y npm -v

    Paso 2. Instalar Angular CLI

    La CLI (Command Line Interface) de Angular permite crear y gestionar proyectos fácilmente. Para instalarla, ejecuta el siguiente comando:

    npm install -g @angular/cli

    Este comando instala la CLI de forma global en tu sistema. Tardará unos minutos.

    Paso 3. Comprobar la instación ejecutando comandos CLI de Angular

    Luego comprueba ejecutando el siguiente comando que se ha instalado correctamente: 

    ng --version

    Deberías ver una salida que incluye la versión de Angular CLI, Node.js y otros paquetes relacionados.

    Paso 4: Crear un nuevo proyecto Angular

    Una vez instalada la CLI, crea un nuevo proyecto con:

    ng new nombre-del-proyecto

    Durante el proceso, se te preguntará:

    ¿Desea habilitar la representación del lado del servidor (ssr) y la generación de sitios estáticos (ssg/prerendering)? Responde Yes si deseas configurar rutas desde el inicio.
    ¿Qué formato de hoja de estilo te gustaría usar? Elige entre CSS, SCSS, SASS o LESS.

    Versiones necesarias

    1. Angular requiere una versión específica de Node.js para funcionar correctamente. Actualmente, se recomienda:

    • Node.js >= 16.14.0.

    Es buena práctica instalar una versión LTS (Long Term Support), ya que es más estable.

    2. NPM (Node Package Manager) incluido con Node.js. Asegúrate de tener una versión actualizada.

    • NPM >= 6.11.0.

    3. Angular CLI

    • Instala la última versión estable de Angular CLI.

    4. TypeScript, Angular requiere una versión específica de TypeScript. La versión mínima recomendada dependerá de la versión de Angular que estés instalando. Por ejemplo

    • Angular 16 requiere TypeScript 5.0+.

    • Angular 15 requiere TypeScript 4.8+.

    5. Navegador moderno

    • Angular funciona en navegadores modernos como Chrome, Firefox o Edge.

    Rutas para tener en cuenta

    • Los proyectos se crean en cualquier carpeta, pero evita espacios y caracteres especiales en las rutas.

    Variables de entorno

    • No es necesario configurar variables especiales para un proyecto base.

    Se necesita de base de datos?

    • No es obligatorio para proyectos simples. Si trabajas con APIs, puedes integrar bases de datos como MySQL, MongoDB, etc.

    Lo esencial es tener configurados los requisitos previos (Node.js y npm) y la instalación de Angular CLI

    Resumen de comandos útiles

    Comando Descripción
    ng version Verifica la versión de Angular CLI instalada.
    ng new nombre-proyecto Crea un nuevo proyecto Angular.
    ng serve Inicia el servidor de desarrollo.
    ng build Compila el proyecto para producción.
    ng generate component nombre Crea un nuevo componente Angular.
    ng generate service nombre Crea un nuevo servicio Angular.
    ng lint Ejecuta el linter para verificar errores en el código.
    ng test Ejecuta las pruebas unitarias configuradas.
